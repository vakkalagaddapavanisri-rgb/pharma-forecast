**Pharma Forecast** is an intelligent, modern web application designed to help pharmacies and healthcare professionals predict medicine demand. By visualizing historical sales data and AI-powered forecasts, it enables proactive inventory management, identifies potential stockouts, and suggests reorder quantities to ensure critical medications are always available.

## ✨ Features

- **📊 Interactive Dashboard:** A comprehensive overview with KPI cards for total demand, revenue, and risk alerts.
- **🔮 AI-Powered Demand Forecasts:** Visualize 30-day predictions with confidence intervals for individual medicines.
- **📈 Category Analytics:** See aggregate demand trends across different medicine categories (e.g., Antibiotics, Pain Relief).
- **⚠️ Reorder Recommendations:** Get a dynamic list of products that are projected to fall below safe stock levels, with suggested order quantities.
- **🔍 Smart Filtering:** Quickly search for medicines and filter the catalog by category.
- **🌗 Dark/Light Mode:** Automatically adapts to your system's preferred color scheme.
- **📱 Responsive Design:** Optimized for desktops, tablets, and mobile devices.

## 🛠️ Tech Stack

- **[Next.js 16 (App Router)](https://nextjs.org/):** React framework for building the application.
- **[React 19](https://react.dev/):** The core library for building the user interface.
- **[Tailwind CSS 4](https://tailwindcss.com/):** A utility-first CSS framework for rapid and consistent styling.
- **[shadcn/ui](https://ui.shadcn.com/):** Re-usable, accessible, and customizable UI components built on top of Tailwind CSS.
- **[Recharts](https://recharts.org/):** A composable charting library for visualizing data.
- **[TypeScript](https://www.typescriptlang.org/):** A strongly typed superset of JavaScript for improved developer experience and code quality.
